# Gokul B - Data Science Portfolio

Welcome to my personal portfolio repository! This simple, modern, and fully responsive website highlights my skills, experience, and projects in Data Science, Machine Learning, and Business Intelligence.

The live website can be found via GitHub Pages (if deployed).

## Overview

- **Built With**: HTML5, Vanilla CSS3 (Custom Design System), and Vanilla JavaScript
- **Features**: 
    - Custom ambient dark theme with vibrant gradient accents
    - Dynamic Canvas API background with floating particles
    - Smooth scroll-reveal animations using `IntersectionObserver`
    - "Download CV" link and a fully functional "Contact Me" form mapped directly to Google Forms (No backend required)

## Key Sections

1. **Hero**: Quick introduction and summary with Download CV link.
2. **About Me**: Educational background and professional experience timeline.
3. **Skills & Expertise**: Categorized technical tags for programming, data analytics, ML/CV, and tools, as well as a Certifications area.
4. **Projects**: Showcase of key projects (`AI Revenue Intelligence Platform`, `Waste Classification App`, `Netcom CRM Analysis`) with links to their respective GitHub repos.
5. **Services**: High-level offerings in Data Analytics, Predictive Modeling, and Deep Learning.
6. **Contact**: A direct message form hooked into Google Forms.

## How to Run Locally

Since this is a static webpage built with raw HTML/CSS/JS, no build steps are required. 

1. Clone the repository:
   ```bash
   git clone https://github.com/Gokulkannan22/gokul_portfolio.git
   ```
2. Navigate into the folder:
   ```bash
   cd gokul_portfolio
   ```
3. Open `index.html` in any modern web browser or use an extension like VS Code's "Live Server".
